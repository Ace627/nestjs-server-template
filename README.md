# NestJS Server Template

## 项目安装

```bash
git clone https://github.com/Ace627/nestjs-server-template.git

cd nestjs-server-template

pnpm install

pnpm start:dev
```

## 目录参考

```bash
|-src
| |-config                              # 环境变量配置目录
|   |-config.development.ts             # 测试环境配置文件
|   |-config.production.ts              # 正式环境配置文件
|-types
| |-env.d.ts                            # 环境变量类型接口定义文件
```
